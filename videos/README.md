# Dossier Vidéos

Placez vos fichiers vidéo ici pour faciliter l'accès depuis l'application.

## Formats supportés

- MP4 (.mp4)
- AVI (.avi)
- MOV (.mov)
- MKV (.mkv)
- WMV (.wmv)
- FLV (.flv)

## Utilisation

1. Placez votre fichier vidéo dans ce dossier
2. Lancez l'application
3. Menu "Fichier" → "Ouvrir une vidéo…"
4. Sélectionnez votre fichier vidéo
5. Activez MediaPipe pour contrôler la vidéo avec les gestes

## Contrôles gestuels

- **Main plate** = Play/Pause
- **Index vers la droite** = Avancer de 2 secondes
- **Index vers la gauche** = Reculer de 2 secondes
